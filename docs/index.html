Share with friends
Friends open this link and auto-connect to your server. No typing needed.
Why host? Your server = low ping for you. Friends nearby also get better ping than Render (Oregon).
Setup (5 min, one time):
1. Install Node.js
2. In a terminal:
git clone https://github.com/gravityst/snake.io.git
cd snake.io
npm install
Run each time you play:
SUBDOMAIN=yourname npm run host
(Windows: set SUBDOMAIN=yourname && npm run host)

Your URL will be https://snakeio-yourname.loca.lt
Then: Paste the URL above → Save. It's saved forever — next time you host, game auto-switches.
